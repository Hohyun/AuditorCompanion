문서 수집 (Collector) 사용법

■ Collector Tab 선택

  • Stage : BEFORE_STARTED (우하단에 표시됨)

가. Case 만들기

  • New 버튼을 눌러 Case 등록
  • Case Name 입력, Auditor Name 입력, 언어 선택
    - 언어 : 인덱스 생성 및 검색시 사용할 분석기 선택 용도
    - Case Directory에 동 정보를 담은 Case 파일 (확장명 case)이 생성됨
  • 성공적으로 등록되면 화면에 Case 명과 Case Directory가 표시됨
  • Stage: CASE_CREATED

나. 파일 검색

  • 좌측 트리에서 검색할 위치, 검색한 파일종류를 선택
  • Analyze 버튼은 눌러 검색 수행
  • 화면에 검색결과가 나타나며, Case Directory에 FileList.txt가 생성됨
  • Stage: ANALYZE_COMPLETED

다. 복사

  • 화면에서 복사를 원하는 파일 선택
  • Checkbox 이용 건별 선택, Shift 키를 이용한 다중 선택 가능
  • Copy 버튼을 누르면, Case_Dir\Files에 해당 파일이 복사됨
  • Stage: COPY_COMPLETED

라. Index 만들기

   1) 복사 후 (다 단계 수행 후) 인덱스를 만드는 경우

  • Index 버튼을 누르면 Case_Dir\Files의 파일 전체에 대해 텍스트를 추출하여 인덱스 생성
  • 인덱스 생성위치: Case_Dir\Index
  • Stage : INDEX_CREATED

   2) 복사 없이 (나 단계 수행 후) 인덱스를 만드는 경우

  • Index 버튼을 누르면 화면에 선택된 파일에 대해 텍스트를 추출하여 인덱스 생성
  • 인덱스 생성위치: Case_Dir\Index
  • Stage : INDEX_CREATED

마. Case Load 하기 (메뉴 - 수집 - Load)

  • 나 단계까지 수행 후 중단한 경우, 나중에 다,라 단계 수행가능
  • Stage : CASE_LOADED
