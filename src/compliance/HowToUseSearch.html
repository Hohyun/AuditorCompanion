텍스트 검색 (Searcher) 사용법

■ Searcher Tab 선택

가. 인덱스 열기

  • Open Index 버튼을 눌러 검색하고자 하는 인덱스를 연다.
  • 인덱스 위치, 인덱스 작성시 지정했던 언어, 페이지 당 조회건수 지정한다
  • 인덱스가 성공적으로 열리면 우상단에 인덱스 위치가 표시된다.

나. 검색어 입력

  • 상단의 검색어 입력란에 질의어를 입력한다.

다. 검색

  • Search 버튼을 눌러 검색을 수행한다.
  • 검색결과가 적합도 상위문서부터 아래 테이블이 나타난다.

   ▶ 결과표시

  • 좌측 테이블 : 파일위치, 제목
  • 우측 탭1 (파일내용) : 파일 텍스트 추출 내용 (검색어는 볼드체로 표시)
  • 우측 탭2 (메타정보) : 파일크기, 원본파일 훼손여부, 페이지수, 생성일, 생성자, 최종수정일, 최종수정자 등

   ▶ Navigation 버튼, Page 지정란

  • 검색 시 우선 첫 페이지만 보여주며, 총페이지 수, 총검색건수가 테이블 위쪽에 표시된다.
  • 좌측 테이블 아래의 이동버튼을 클릭 시 검색을 다시 수행하여 원하는 페이지를 보여준다.
  • Page 지정란에 원하는 페이지 번호를 입력한 후 Search 버튼을 누르면 해당 페이지로 직접 이동할 수 있다.

   ▶ 외부 프로그램 이용하여 파일 열기

  • 죄측 결과 테이블의 행을 더블클릭하면 해당 파일을 시스템에 지정된 외부(엑셀, 원드, 파워포인트 등) 프로그램을 이용하여 열어준다.
